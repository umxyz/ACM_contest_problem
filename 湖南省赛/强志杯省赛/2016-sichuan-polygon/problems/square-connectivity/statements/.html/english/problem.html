Road History
time limit per test
1 second
memory limit per test
1024 megabytes
input
standard input
output
standard output

Bobo is studying the history of roads in ICPCCamp. In ICPCCamp, there are n cities with m bidirectional roads. The i-th road connects the ai-th and bi-th cities.

There were no roads initially. Eventually, roads were built in the order 1, 2, ... m. Bobo would like to know the number of pairs of cities which allow an odd drive after the i-th road was built. An odd drive between cities u and v is possible only if there exists v1, v2, ..., v2k for some positive integers k such that v1 = u, v2k = v and there is a road connecting cities vi and vi + 1. Passing by a city more than once is allowed.

Input

The first line contains 2 integers n, m (1 ≤ n, m ≤ 105).

The i-th of the following m lines contains 2 integers ai, bi (1 ≤ ai, bi ≤ n).

Output

m lines with integers w1, w2, ..., wm where wi denotes the number of pairs allowing an odd drive after the i-th road was built.

Examples

Input
3 3
1 2
2 3
3 1
Output
1
2
3
Input
4 3
1 2
3 4
2 3
Output
1
2
4
