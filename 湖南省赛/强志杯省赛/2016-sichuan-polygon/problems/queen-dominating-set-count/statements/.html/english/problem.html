2 seconds

1024 megabytes

In ICPCCamp, there is a chessboard with n rows and m columns.

Bobo places k distinguishable queens in k different cells on the chessboard. There are different configurations where

If ci is the number of cells attacked by at least one queen in the i-th configuration, find out (c1 + c2 + ... + ct) modulo (109 + 7).

Note that a queen can attack all cells in the same row, column and diagonal including the cell she stands on.

Input

3 integers n, m, k (1 ≤ n, m ≤ 109, 1 ≤ k ≤ min{n × m, 8}).

Output

An integer denotes (c1 + c2 + ... + ct) modulo (109 + 7).

Examples

Input
2 2 2
Output
24
Input
8 8 8
Output
723759469
